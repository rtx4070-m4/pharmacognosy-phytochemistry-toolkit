🌿 Pharmacognosy & Phytochemistry Toolkit










A standalone scientific web toolkit for performing pharmacognosy and phytochemistry calculations used in herbal drug evaluation and natural product analysis.

This lightweight application provides essential laboratory calculations commonly used in pharmaceutical research, pharmacognosy labs, herbal drug standardization, and phytochemical analysis.

The toolkit is built as a single-file responsive web application with no external dependencies, making it easy to run on any device or browser.

🔬 Overview

Pharmacognosy laboratories frequently perform repetitive calculations for evaluating crude drugs, plant extracts, and herbal formulations. This toolkit centralizes those calculations into a clean, fast, and user-friendly interface.

It enables researchers and students to quickly compute:

Extract yield percentages
Moisture content
Ash values
Swelling index
Hemolytic index
Bitterness value
Extractive values

The application is designed to be portable, offline-capable, and extremely lightweight, allowing use in labs, classrooms, or field research environments.

🧮 Calculators Included
1️⃣ Extract Yield Calculator

Determines the percentage yield of extract obtained from crude plant material.

Formula:

Extract Yield (%) = (Weight of Extract / Weight of Plant Material) × 100
2️⃣ Moisture Content Calculator

Used to determine water content in crude drug samples.

Formula:

Moisture Content (%) = ((Initial Weight − Final Weight) / Initial Weight) × 100
3️⃣ Ash Value Calculator

Evaluates inorganic residue present in crude drugs.

Includes calculations for:

Total Ash
Acid-Insoluble Ash
Water-Soluble Ash
4️⃣ Swelling Index Calculator

Measures the volume increase of plant materials in aqueous medium.

Used for evaluating mucilage-containing drugs such as:

Ispaghula
Linseed
Psyllium
5️⃣ Hemolytic Index Calculator

Estimates the hemolytic activity of saponin-containing plant materials.

Used in pharmacognosy labs to evaluate membrane-lytic potential.

6️⃣ Bitterness Value Calculator

Determines bitterness relative to quinine hydrochloride standard.

Used for drugs like:

Gentian
Quassia
Andrographis
7️⃣ Extractive Value Calculator

Determines the amount of active constituents extracted using solvents.

Common extractive values:

Alcohol soluble extractive
Water soluble extractive
Ether soluble extractive
